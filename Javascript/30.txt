## üå± What is Async / Await?

üëâ `async` and `await` are **keywords** introduced in ES2017 (ES8)
üëâ They are **syntactic sugar** over **Promises** ‚Äî meaning they make Promises easier to write and read.
üëâ They allow you to write **asynchronous code that looks synchronous**.

---

### üß© Analogy:

Imagine you order food üçî online.

* You **place the order** ‚Üí async function starts
* You **wait** for food to arrive ‚Üí `await`
* Once delivered, you **eat** ‚Üí continue next line of code

---

## ‚öôÔ∏è Basic Syntax

```js
async function myFunction() {
  let result = await somePromise();
  console.log(result);
}
```

* `async` ‚Üí makes the function return a **Promise**.
* `await` ‚Üí pauses the function until the **Promise** is resolved or rejected.

---

## üîÑ Example 1 ‚Äî Without Async/Await

```js
function getData() {
  fetch("https://jsonplaceholder.typicode.com/users")
    .then(response => response.json())
    .then(data => console.log(data))
    .catch(err => console.log(err));
}
getData();
```

---

## ‚ú® Example 2 ‚Äî With Async/Await (Cleaner)

```js
async function getData() {
  try {
    let response = await fetch("https://jsonplaceholder.typicode.com/users");
    let data = await response.json();
    console.log(data);
  } catch (error) {
    console.log("Error:", error);
  }
}
getData();
```

‚úÖ Easier to read
‚úÖ Looks like synchronous code
‚úÖ Still non-blocking!

## üß† Important Rules

1. You can **only use `await` inside `async` functions**.
2. `async` functions **always return a Promise**.
3. Use **`try...catch`** to handle errors.
4. You can await multiple Promises together using `Promise.all()`.

---

## ‚öôÔ∏è Example 3 ‚Äî Multiple Awaits

```js
async function example() {
  let user = await fetch("https://jsonplaceholder.typicode.com/users/1");
  let post = await fetch("https://jsonplaceholder.typicode.com/posts/1");

  let userData = await user.json();
  let postData = await post.json();

  console.log(userData, postData);
}
example();
```

Here, the second fetch waits for the first to finish.
To run both **parallelly**, use üëá

```js
async function exampleParallel() {
  const [user, post] = await Promise.all([
    fetch("https://jsonplaceholder.typicode.com/users/1"),
    fetch("https://jsonplaceholder.typicode.com/posts/1")
  ]);

  console.log(await user.json(), await post.json());
}
exampleParallel();
```

## üí° In Short

| Concept            | Description                          |
| ------------------ | ------------------------------------ |
| **async**          | Makes a function return a Promise    |
| **await**          | Waits for a Promise to resolve       |
| **try/catch**      | Handles errors like `.then/.catch()` |
| **Cleaner syntax** | Easier to read and maintain          |
